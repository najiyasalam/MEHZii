# MEHZii

This website is for MEHZII, a brand based in Kerala, India, that specializes in natural, healthy snacks and food products.

Core Products: They primarily sell honey-infused dry fruits, wholesome nuts, and aromatic spice blends.

Key Selling Points: The brand emphasizes 100% natural ingredients with no artificial preservatives or additives. Their products are marketed as being ethically sourced and handcrafted.

Target Audience: Health-conscious individuals looking for nutrient-rich snacks or premium food gifts.

Founder: The company was founded by Swadiq Ali and operates out of Perinthalmanna, Malappuram.

## 📦 Installation & Local Development

1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/najiyasalam/MEHZii.git](https://github.com/najiyasalam/MEHZii.git)
    ```
2.  **Navigate to the folder**:
    ```bash
    cd MEHZii
    ```
3.  **Open the project**:
    Simply open `index.html` in your favorite web browser or use a "Live Server" extension in VS Code.
